#### 1.2.5. git插件

- git history
  - 左侧导航栏显示文件历史（右键，时间线）
  - 选择显示时间线/右上角时钟
- gitlens
  - 代码悬停后显示浮标，最后修改信息
  - 鼠标指向显示位置，可以查看查分信息

- draw.io(下载量最大的)
  - 绘制简单图表
